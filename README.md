# AirPulse

Aplikasi monitoring kualitas udara berbasis Flutter untuk Tugas Akhir mata kuliah Teknologi dan Pemrograman Mobile.

## Fitur Utama

- **Dashboard AQI** — Data kualitas udara real-time dari Open-Meteo API
- **Peta Interaktif** — Peta kualitas udara dengan pencarian kota dan long-press
- **Perencana Aktivitas** — Prakiraan AQI 24 jam dengan notifikasi terjadwal
- **Konversi** — Konversi mata uang dan zona waktu global
- **AirBot** — Chatbot AI berbasis Gemini untuk konsultasi kualitas udara
- **Kuis** — Mini game edukasi tentang polusi udara
- **Keamanan** — Login terenkripsi SHA-256 dan autentikasi biometrik
- **Sensor** — Gyroscope (shake-to-refresh) dan Accelerometer (deteksi aktivitas lari)
